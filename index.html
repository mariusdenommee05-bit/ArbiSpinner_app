LIVE Sharp money tracker · First mover detection · CLV analysis
How steam detection works: Sharp sportsbooks (Pinnacle, Circa) move their lines first when professional bettors place large wagers. When we detect a line move at a sharp book before soft books react, that's a steam move — a signal that the market is mispriced and an arb or value bet window is about to open.

🛠 Developer: What you need before selling subscriptions ▼ Show
This file is frontend-only — after payment, there's no server to verify who paid. Anyone could share the HTML and unlock all features. To properly gate features behind subscriptions, you need 3 backend components:
1️⃣ Auth (Login)
Users need accounts so you know who paid. Use Supabase Auth (free, email + Google OAuth) or Firebase Auth. ~1 day to wire up.
2️⃣ Stripe Webhooks
Stripe fires checkout.session.completed to your server after payment. Your server writes the user's plan to the DB. Without this, payment is unverifiable.
3️⃣ Plan Gating
On login, fetch user's plan. Pass it to the frontend to unlock features. Store the Odds API key server-side — never expose it in HTML.
💡 Fastest free stack to launch: Supabase (auth + DB) + Vercel (hosting + serverless webhook handler) + Stripe (payments). Cost to launch: $0/mo until you scale. Estimated setup time with AI assistance: 1–2 days.
